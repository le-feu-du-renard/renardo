# renard'o — Dryer Controller

Food dehydrator controller built on the Raspberry Pi Pico W. Hybrid heating (hydraulic + electric), RS485 sensors, physical panel interface, SD card session logging.

## Project Philosophy

**Low-tech, repairable, and accessible.** No touchscreen, no software menu — everything is physical: potentiometers, selector switch, buttons, panel voltmeters, and indicator LEDs.
